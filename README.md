# UAV-OMVCD
无人机倾斜多视角全景变化检测数据集（Unmanned Aerial Vehicle-Oblique Multi-View Panoramic Change Detection Dataset, UAV-OMVCD）。该数据集是一款面向HRST-MSLC的计算机视觉数据集，在合肥、唐山、南京等典型区域开展了多期次倾斜全景影像采集。针对硬化地表、工程机械、建筑物、温室大棚及堆土等关键地物，系统记录了其在不同时间节点下的结构增减、形态演变及缺陷演化等季节性变化特征。突破了传统正射影像垂直视角的局限，通过多角度倾斜观测有效覆盖了地物侧立面及隐蔽区域，同时克服了季节性伪变化干扰。
